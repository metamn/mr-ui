## Typographic Grid

- Sets up a basic typographic grid with `font-size:100%; line-height:1.25` in the `<body>`
- Defines the grid size global variable `--lem: 1.25em`
- Displays grid lines

## Usage

```Javascript
class App extends Component {
	render() {
		return (
			<>
				<TypographicGrid/>
				...
			</>
		)
	}
}
```
